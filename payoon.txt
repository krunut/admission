Super admin ระดับเขต
	สร้าง Admin โรงเรียบ
	กำหนดวัน เปิด / ปิด ระบบ 
	ดูรีพอร์ต ทุกโรงเรียน
Admin โรงเรียน
	จัดการ user ของแต่ละโรงเรียน
	จัดการข้อมูลการรับสมัร
		วันรับ (เช้าบ่าย) / จำนวน
		วันเวลาสอบ / วันเวลารายงานตัว / วันเวลามอบตัว
		กำหนดห้องสอบ  (จำนวนที่นั่งในแต่ละห้อง)  (เรียงได้) ใช้ในการกำหนดวันสอบอัตโนมัติ แต่ละประเภท
		กำหนดวิชา/วิธีสอบ/คะแนนเต็ม (แต่ละประเภท)
		อื่นๆ http://122.154.19.105/schoolict/rub_student/
	ออกใบสมัคร (บันทึกข้อมูล)
	แก้ไขข้อมูลใบสมัคร (ตลอดเวลา)
	ล้างเลขใบสมัคร (อย่าปล่อยที่ว่าง)
	บันทึกผลสอบ (กรอก) (อัพไฟล์) / ผ่านไม่ผ่าน / ผ่าน สำรอง
	บันทึกรายงานตัว วันเวลา
	บันทึกมอบตัว
	ดูรีพอร์ต โรงเรียน ตัวเอง
		Export เป็น excel / PDF
	
User โรงเรียน
	ออกใบสมัคร
	ปริ้น
	แก้ไขได้ จนกว่าออกใบสมัคร
	เลือกสถานะ (ประเภท, นอกเขต ในเขต ในเขตไม่สมบูรณ์) จนกว่าออกใบสมัคร
Registrant
	แจ้งความจำนง
	แก้ข้อมูลผู้สมัคร จนกว่าออกใบสมัคร
Public
	สถิติทั่วไป
